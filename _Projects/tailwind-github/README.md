https://gitee.com/MASUKA/tailwind_github_ui

## Setup

https://tailwindcss.com/docs/installation

```sh

npm install -D tailwindcss
npx tailwindcss init
```

Modify tailwind.config.js

Build

```sh
npx tailwindcss -i index.css -o output.css --watch
```
